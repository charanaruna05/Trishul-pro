# 🔱 Trishul Pro - Precision Trading Dashboard

**Trishul Pro** एक शक्तिशाली स्टॉक मार्केट ट्रेडिंग डैशबोर्ड है, जिसे ट्रेडर्स को बाज़ार में त्रिशूल जैसी सटीक पकड़ देने के लिए बनाया गया है। यह तेज़ डेटा और सुरक्षित एपीआई इंटीग्रेशन के साथ काम करता है।

## ✨ मुख्य विशेषताएं (Features)

* **Trishul Speed:** बाज़ार की हर हलचल पर पैनी नज़र।
* **Secure Trading:** आपकी एपीआई (API) और डेटा पूरी तरह सुरक्षित।
* **Advanced Integration:** Angel One और Shoonya ब्रोकर्स के साथ आसानी से कनेक्ट।
* **Hybrid App:** वेब पर इस्तेमाल करें या मोबाइल के लिए APK बनाएँ।

## 📁 प्रोजेक्ट स्ट्रक्चर (Folders)

आपका कोड व्यवस्थित है:
- `www/` : आपका मुख्य घर (Root Folder)
- `www/css/` : स्टाइल और लुक।
- `www/js/` : त्रिशूल प्रो का असली दिमाग (App Logic & API)।
- `www/index.html` : ऐप का मुख्य द्वार।

## 🚀 सेटअप कैसे करें (Setup Guide)

1. अपनी रिपॉजिटरी को क्लोन करें।
2. `www/js/config.js` में अपनी निजी **API Key** जोड़ें।
3. `www/index.html` को खोलें और ट्रेडिंग शुरू करें!
