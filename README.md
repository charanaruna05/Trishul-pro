# 🔱 Trinetra Pro - Trading Dashboard

**Trinetra Pro** एक आधुनिक स्टॉक मार्केट ट्रेडिंग डैशबोर्ड है जिसे व्यापारियों (Traders) के लिए तेज़ और सटीक डेटा एनालिसिस प्रदान करने के लिए बनाया गया है। यह वेब और मोबाइल (APK) दोनों प्लेटफॉर्म पर काम करता है।

## ✨ मुख्य विशेषताएं (Features)

* **Real-time Data:** लाइव मार्केट डेटा फीड।
* **Fast Execution:** तेज़ और सुरक्षित ट्रेडिंग अनुभव।
* **Multi-Broker Support:** Angel One और Shoonya जैसे प्रमुख ब्रोकर्स के साथ इंटीग्रेशन।
* **Mobile Optimized:** पूरी तरह से रिस्पॉन्सिव डिज़ाइन, जो मोबाइल पर APK के रूप में बेहतरीन चलता है।

## 📁 फोल्डर स्ट्रक्चर (Project Structure)

आपका कोड निम्नलिखित तरीके से व्यवस्थित है:
- `www/` : मुख्य सोर्स फोल्डर।
- `www/css/` : डिज़ाइन और स्टाइलिंग फाइलें।
- `www/js/` : ऐप का लॉजिक, चार्ट और API कॉन्फ़िगरेशन।
- `www/index.html` : मुख्य एंट्री पॉइंट।

## 🚀 कैसे शुरू करें (How to Run)

1. इस रिपॉजिटरी को क्लोन या डाउनलोड करें।
2. `www/js/config.js` में अपनी **API Key** और क्रेडेंशियल सेटअप करें।
3. `www/index.html` को किसी भी ब्राउज़र में खोलें या इसे APK में बदलें।

## 🛠 तकनीकी जानकारी (Built With)

* **Frontend:** HTML5, CSS3, JavaScript
* **Deployment:** GitHub & Vercel
* **App Framework:** Capacitor / WebView
